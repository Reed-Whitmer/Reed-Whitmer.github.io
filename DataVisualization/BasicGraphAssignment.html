###Create the double bar graph
labels = df_opiod['CountyName']
x = np.arange(len(labels))
width = 0.35

fig, ax = plt.subplots()
rects1 = ax.bar(x - width/2, df_opiod['FFHO-Rate'] , width, label='Heroine')
rects2 = ax.bar(x + width/2, df_opiod['FPOO-Rate'], width, label='Opiod')

fig.suptitle('Opiod Overdoses vs Heroine Overdoses by county')
ax.xaxis.set_label_text('Counties')  
ax.yaxis.set_label_text('Overdoses per 100,000 residents')
ax.set_xticks(x)
ax.set_xticklabels(labels)
ax.legend()

ax.autoscale(tight=True)
plt.xticks(rotation = 90)
fig.set_size_inches(40,5)

plt.show()

I chose a doublebar graph because the user will be able to check each county at the same time and look at the direct comparison between heroin and opiods overdose.
